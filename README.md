
# Recetas de Odin

Es un proyecto de una pagina web estatica que proporciona una interfaz amigable de varias Recetas.Su funcion principal ser fácil de usar y que esta diseñada para amantes de la cocina. Explora una extensa colección de recetas, desde platos cotidianos hasta elaboradas creaciones culinarias.
